# Estelar Habits ✨

O **Estelar Habits** é uma aplicação de gerenciamento e monitoramento de hábitos, projetada para auxiliar na produtividade e organização pessoal. 

Esta versão representa a evolução técnica do projeto, que foi originalmente concebido em PHP e MySQL e agora está sendo refatorado para uma arquitetura moderna focada em performance, escalabilidade e uma experiência de usuário (UX) superior.

## 🚀 Tecnologias e Ferramentas

Para esta nova fase do projeto, foram selecionadas tecnologias de ponta no ecossistema Web:

* **React**: Biblioteca para construção de interfaces modulares e reativas.
* **Vite**: Ferramenta de build de última geração para um desenvolvimento extremamente rápido.
* **Tailwind CSS**: Framework CSS utilitário para um design responsivo e minimalista.
* **Node.js & npm**: Ambiente de execução e gerenciamento de dependências.

## 📋 Pré-requisitos

Para rodar este projeto localmente, você precisará de:
* [Node.js](https://nodejs.org/) (Versão LTS recomendada).
* Um editor de código (como o [VS Code](https://code.visualstudio.com/)).

## 🔧 Como Iniciar o Projeto

Siga os passos abaixo para configurar o ambiente de desenvolvimento:

1. Aceder ao diretório
No terminal, navegue até a pasta raiz do projeto:
cd C:\\xampp\\htdocs\\estelar_habit_tracker1.2

2. Instalar as dependências
Baixe todos os pacotes necessários listados no package.json:
npm install

4. Iniciar o servidor de desenvolvimento
Execute o comando para subir o servidor local do Vite:
npm run dev


## 🛠️ Comandos Disponíveis
npm run dev: Inicia o ambiente de desenvolvimento.
npm run build: Cria a versão de produção otimizada na pasta /dist.
npm run preview: Testa o build de produção localmente.
